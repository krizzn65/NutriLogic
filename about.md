1. Posisi NutriLogic (dibanding MoriSmart lama)

Bukan cuma pencatat berat/tinggi (CRUD) → tapi asisten gizi digital.

Fokus ke dua hal utama:

Smart Monitoring tumbuh kembang (otomatis hitung status gizi, Z-score WHO).

Digital Nutrition / Nutri-Assist (bantu ortu ambil keputusan soal makanan, bukan cuma input data).

2. Fitur Utama untuk ORANG TUA (User Publik)

Fokus: monitoring + edukasi + dorong perilaku baik

Dashboard Pertumbuhan Anak

Grafik KMS digital → update otomatis setiap ada data baru.

Status gizi (pendek/sangat pendek/normal) dihitung otomatis (bukan diketik manual).

Nutri-Scanner / Nutri-Assist (Inovasi Kunci)

Ortu input bahan makanan yang ada di rumah.

Sistem kasih rekomendasi menu/MPASI yang sesuai gizi anak.

Menguatkan aspek “Digital Nutrition”.

Gamifikasi “Stunting Fighter”

Poin & badge untuk:

rutin update data,

baca artikel edukasi,

patuh jadwal posyandu.

Tujuan: bikin ortu betah & engaged.

Konsultasi / Chat dengan Kader

Tanya jawab singkat seputar gizi, jadwal posyandu, keluhan anak.

Reminder Kesehatan

Notifikasi imunisasi, vitamin, jadwal posyandu.

Bisa lewat WA / in-app (minimal konsep/simulasi dulu).

3. Fitur Utama untuk KADER / ADMIN POSYANDU

Fokus: efisiensi kerja + analitik + deteksi dini

Manajemen Data Posyandu

CRUD data anak, orang tua, posyandu, jadwal.

Satu panel untuk kelola semua.

Input Massal (Bulk Input) Hari H Posyandu

Form atau tampilan khusus untuk input banyak anak sekaligus (hemat waktu di lapangan).

Early Warning System

Sistem auto-flag anak berisiko:

grafik turun,

status gizi memburuk,

jarang datang posyandu / jarang update.

Muncul di dashboard prioritas untuk intervensi cepat.

Reporting & Export

Laporan bulanan/periodik ke PDF/Excel.

Bisa disetor ke puskesmas/desa.

Broadcast Informasi

Kirim info jadwal posyandu, pengumuman penting ke semua orang tua terdaftar.
